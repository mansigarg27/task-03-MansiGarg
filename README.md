# Password Generator

## Description

This is a simple Python program that generates a random password based on the length entered by the user. The generated password contains uppercase letters, lowercase letters, and numbers to make it stronger and more secure.

## Features

* Takes password length as input from the user
* Generates a random password
* Includes uppercase letters, lowercase letters, and digits
* Ensures better password complexity

## Technologies Used

* Python
* Random Module

## How It Works

1. The user enters the desired password length.
2. The program selects random characters from uppercase letters, lowercase letters, and numbers.
3. The characters are shuffled to increase randomness.
4. The final password is displayed on the screen.

## Sample Output

Enter password length: 8

Generated Password: A7kP2xQ9
